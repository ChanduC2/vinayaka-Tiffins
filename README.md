# Vinayaka Tiffins – Online Food Ordering Website

A responsive web application for ordering breakfast items online.

## Features
- Interactive food menu
- Shopping cart system
- UPI payment integration
- QR code payment
- Order confirmation system
- Google Maps location
- Contact form

## Technologies Used
- HTML
- CSS
- JavaScript
